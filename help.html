What Is Markdown?

Markdown is a plain text formatting syntax designed so that it can optionally be converted to HTML using a tool by the same name.
Markdown is popularly used as format for readme files, or for writing messages in online discussion forums,
or in text editors for the quick creation of rich text documents.

  • Indent your code to display a code block
  • Have extra return spaces between paragraphs
  • Use # for h1, ## for h2 and ### for h3, etc.
  • Inline code are surround by `backtick` key.
